# Calendar widget

> Simple calendar widget built on Vue.js.

See [demo](http://otouto.github.io/vue-calendar/).

Calendar supports keyboard:
- *arrows* - change date;
- *enter* - choose date;
- *space* - toggle view;
- *esc* - close calendar.
 

##Installation

```
git clone
npm i
node app
```

Open [http://127.0.0.1:3005/](http://127.0.0.1:3005/)
